## 🎯 Features

- **Task Management** — Create, assign, and track tasks with status updates
- **Team Collaboration** — Assign tasks to team members with role-based access
- **Project Dashboard** — Visual overview of project progress and team workload
- **Real-time Updates** — Instant task status changes and notifications
- **User Authentication** — Secure login system for team members

## 🚀 Getting Started

### Prerequisites
- Any modern web browser
- Local server (e.g., Live Server, XAMPP, or Node.js server)

### Installation

1. Clone the repository
```bash
git clone https://github.com/Raj-Kori/Project-Management-tool.git
cd Project-Management-tool
```

2. Extract `Project_management_system.zip`

3. Open `index.html` in your browser or run via local server
